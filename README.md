## 📝 Project Overview
This project aims to build a machine learning model to detect fraudulent credit card transactions. The dataset contains transactions made by credit cards in 2023, where each transaction is labeled as either fraudulent or legitimate.

## 📂 Repository Structure
- `notebooks/`: Contains the Jupyter Notebook with data analysis, preprocessing, and modeling.
- `dataset/`: Contains the raw transaction data (CSV).

## 📊 Dataset Description
The dataset includes the following features:
- **id**: Unique identifier for each transaction.
- **V1-V28**: Anonymized features representing various transaction attributes.
- **Amount**: The transaction amount (scaled during preprocessing).
- **Class**: Binary label (1 for Fraud, 0 for Legitimate).

## 🚀 How to Run
1. Clone the repository.
2. Ensure you have `pandas`, `scikit-learn`, and `matplotlib` installed.
3. Open the notebook in `notebooks/projeto_tcc_credit_card_fraud_1.ipynb` and execute the cells.
